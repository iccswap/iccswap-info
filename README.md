# Iccswap History


Transaction and liquidity pool stats for [Iccswap](https://app.iccswap.com/#/swap).

Check it out live: [https://www.iccswap.com/home](https://www.iccswap.com/home).

### To Start Development

###### Installing dependencies
```bash
yarn
```

###### Running locally
```bash
yarn start
```
